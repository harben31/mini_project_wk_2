CSS Snippet Cheatsheet

Ever have trouble remembering the syntax for your favorite CSS code? Give it a perminent home and add it to this page. Select any snippet below and it'll automatically select all the code for you to copy.



Flexbox Row

Use these three properties to create a Flexbox Row layout

.row { display: flex; flex-direction: row; flex-wrap: wrap;

Flexbox Column

Use this to create a Flexbox Column layout

.row { display: flex; flex-direction: column; }

CSS Grid layout

Build a twelve-column layout using CSS Grid

.grid { display: grid; width: 100%; grid-template-columns: repeat (12, 1fr); }

Linear Gradients

This will create a linear gradient from top to bottom

.linear-gradient-background { background-image: linear-gradient( rgba(232, 102, 236, 0.3) 0%, rgba(232, 102, 236, 0.6) 100% ); }

Box Transition Glow

Use transitions and box shadows to glow on hover

.code .code-card { border-radius: 8px; transition: all 0.5% ease-in-out; } .code-card:hover, .code-card:hover, .card-header { box-shadow: inset 0px 0px 8px rgba(232, 102, 236, 1) rgba(232, 102, 236, 1); }

Overlay card with title

Use position properties and negative margins to raise elements higher than their natural starting point.

.card-header { position: relative; margin-top: -20px; }
